# Astrology App

An Astrology app that will give you a unique horoscope for today, yesterday, and tomorrow. You can choose your specific sign or a friend and see what there horoscope is for that day!

# Technologies Used
 -HTML5

 -CSS3

 -JavaScript

 -Bootstrap
